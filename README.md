## 2. PC 상세정보

PC 사양과 실시간 사용률을 카드 형태로 상세히 보여주고, 서비스에 필요한 입력·업데이트 작업을 제공합니다.

### 2-1. 시스템 정보
- 제조사 · 모델명 · 제조번호 표시, 각각 **Copy** 버튼으로 복사/붙여넣기
- **모델/제번입력 (Run)** — 메인보드 교환 후 모델명·제조번호를 원클릭 입력
- **로고변경 (Unlock / Run)** — 특정 ODM 모델 로고변경 원클릭 입력 (대상 모델이 아닐 경우 "대상 모델 아님" 팝업)

### 2-2. BIOS
- BIOS 버전 · BIOS 날짜 · 메인보드 모델 표시
- **BIOS Update**
  - **Online** — 온라인으로 최신 BIOS 확인·업데이트 진행
    - **[변경]** 지원 모델 확대: 2020년 이후 모델뿐 아니라 2016·2017·2018·2019년 레거시 모델까지 지원
    - 현재/최신 버전 비교 + 릴리즈 노트 표시, 최신이면 "현재 최신버전의 BIOS가 적용되어 있습니다" 안내
  - **Offline (Run)** — `NT_WIN_Flash_model` 폴더의 오프라인 BIOS를 원클릭 업데이트 (자동으로 모델 매칭되어 진행)

### 2-3. 운영체제
- OS 이름 · 버전 · 빌드 · 아키텍처 · 마지막 부팅 시각
- **[변경]** '설치일' 항목 제거 (표기 간소화)
- **RDPK (Check / Run)** — DPK Tool(OA3 QA Client / OA3 Service Client) 실행 바로가기

### 2-4. CPU / MEMORY / GPU
- **CPU** — 모델·제조사·코어/스레드·최대 클럭·소켓·L2/L3 캐시 및 실시간 사용률
- **MEMORY** — 총 용량·사용 중·여유·폼팩터·속도·제조사·부품번호 및 실시간 사용률
- **GPU** — 이름·제조사·VRAM·드라이버·해상도·재생률(주사율) 및 실시간 사용률

### 2-5. 저장장치 (Disk)
- 드라이브별 모델명·제조사·타입·용량·S.M.A.R.T. 상태·사용 시간·드라이브 온도·웨어 레벨링 표시
- **불량섹터 검사** — 드라이브 선택 후 빠른검사/정밀검사. 블록별 응답 속도를 색상 히트맵(<40ms ~ >4.8s, Err)으로 시각화하고 속도·검사·완료·ETA·에러 카운트 로그 제공

### 2-6. 네트워크 / 블루투스
- **네트워크** — 어댑터 모델·연결상태·IP·MAC·연결 속도
- **블루투스** — 어댑터·연결상태·IP/MAC·속도

### 2-7. 배터리
- AC 연결/충전 상태, 충전률(%), 전압·전류·전력·RSOC·FCC·설계 용량·배터리 효율·사이클 카운트·배터리 남은 수명·내부 온도(인하우스만)
- 남은 수명 로직은 My gram과 동일 — 인하우스는 EC 데이터, ODM은 배터리 레포트 데이터 로드
- 판정 색상: **정상(60~100%) · 경고(40~60%) · 교체 권장(0~40%)**

## 5. PC 점검하기

각 항목을 클릭하면 개별 점검, 우측 상단 **전체점검**으로 마법사 순차 점검(10개 항목 자동 또는 사용자 확인으로 불량 판단)을 시작합니다.

| 항목 | 내용 |
|---|---|
| Wi-Fi 점검 | 루프백 테스트(게이트웨이/WAN Ping) · 신호강도 측정 · 다운로드/업로드 속도 측정 |
| 블루투스 점검 | 주변 디바이스 신호 감지(1개 이상 검색 시 OK) · 연결 상태 확인 · 장치 수 비교 |
| 키보드 점검 | 모든 키 입력 정상 동작 확인(실시간 테스트), 점검 중 윈도우키 메뉴 차단, PASS/FAIL 판정 |
| 카메라 점검 | 일반 카메라 · 적외선(IR) 카메라 동작 확인, 라이브 프리뷰 출력 |
| 불량화소 점검 | 전체화면 R/G/B 색상 순차 출력으로 육안 점검 |
| LID 스위치 점검 | 노트북 LCD OPEN/CLOSE 센서 동작 테스트 |
| CPU 온도 확인 | 실시간 모니터링(인하우스 EC 정보, ODM 모델은 LHM 우회 감지) |
| FAN 점검 | FAN RPM 강제 구동 테스트(인하우스 EC 정보, ODM 미지원), 싱글/듀얼팬 자동 감지, **[신규]** 팬 수 수동 보정(자동/싱글팬/듀얼팬 버튼, PE 환경 오인지 대응) |
| 스피커 테스트 | 좌/우 채널 개별 출력 테스트, 사운드 정상 여부 확인 |
| 마이크 테스트 | 스피커로 기준음 출력 → 마이크 입력 판단(입출력 루프백 검증) |

## 6. Aging Test **[신규]**

전원 반복 · 디스크 복사(XCopy) · 화면 녹화 에이징을 수행합니다. (Windows 전용)
테스트 종류를 타일로 선택하고 옵션 설정 후 시작합니다.

### 6-1. 전원 반복 (인앱 방식)
- **유형** — 리부트(재시작) / 절전(Sleep S3) 중 선택
- **반복 횟수 · 딜레이(초)** 지정
- **Windows 암호** — 자동 로그인용(리부트 테스트 필수)
  - ※ PIN이 아니라 실제 계정 암호를 입력. PIN만 쓰는 계정은 로그인 옵션에서 "Windows Hello 로그인만 허용"을 끄거나 계정 암호를 설정해야 무인 자동 로그인이 동작
- **동작** — 재부팅/절전을 지정 횟수만큼 자동 반복. 재부팅 유형은 작업 스케줄러 자동 실행 + Winlogon 자동 로그인으로 부팅 후 이어서 진행
- **절전 복귀** — RTC 웨이크 타이머로 자동 기상 + 복귀 직후 디스플레이 자동 점등(화면 꺼짐 방지)
- **완료 처리** — 모든 회차 종료 시 재부팅 후 EULA를 건너뛰고 Aging 화면에 완료 결과를 표시, 자동 로그인·작업 스케줄러·전원 설정을 원상 복구

### 6-2. 디스크 복사 (XCopy)
- **반복 시간(분)** 지정(기본 5분)
- 매 회차 더미 파일 생성 → 복사 → 삭제를 반복해 디스크에 부하
- **진행 로그** — 동작을 10회 단위로 묶어 표시 (예: `#1~10 더미 생성 → 복사 → 삭제 10회 (각 200MB) · 누적 3.9GB`)
- **상단 상태** — 남은 시간 카운트다운 (예: `남은 시간 00:04:32 · 123회 · 24.0GB`)
- 시작 전 디스크 여유 공간 확인, 종료/중지 시 임시 파일 자동 삭제(로그 안내), 디스크 점유는 순간 최대 수백 MB로 제한

### 6-3. 화면 녹화
- **녹화 시간(분)** 지정(0이면 "중지/초기화"를 누를 때까지 계속)
- 증상 재현 시 화면을 .mp4로 녹화해 나중에 확인 (`문서\GAP_Aging` 저장)
- **녹화 폴더 열기** 버튼(화면 녹화 선택 시 노출)으로 저장 폴더 열람
- 녹화 중 앱을 종료해도 파일을 정상 마무리(finalize)한 뒤 종료

## 7. 이벤트 로그 **[고도화]**

Windows System/Application 로그의 위험·오류·경고를 시간대별로 나열하고, 노트북 하드웨어 진단에 맞춰 필터링합니다.

- **기본 표시** — 위험·오류·경고(경고 항상 포함) 최신순
- **필터** — 이벤트 ID(숫자) · 로그 원본(전체/System/Application)
- **[신규] 진단 분류** — 진단 카탈로그로 각 이벤트를 자동 분류
  - 카테고리: 전원·크래시 / 디스크 / CPU·메모리 / 발열·전력 / 그래픽 / 네트워크 / WTG·USB / 소프트웨어 / 노이즈
  - 동일 ID라도 소스별로 의미가 다른 경우(예: 11=Disk vs Win32k, 18=WHEA vs BTHUSB)를 소스로 구분
- **[신규] 진단 이벤트만** — 카탈로그에 매칭된 이벤트만 표시(빠른 트리아지)
- **[신규] 노이즈 숨기기(기본 ON)** — 정상 동작에서도 흔한 항목(DCOM 10016, 서비스 7031/7034, LSA 6155) 숨김
- **목록** — 수준 · 발생 시간 · 로그 · 이벤트 ID · 진단 분류(색상) · 원본 · 설명
- **상세** — 행 선택 시 판별 기준(예: "41 Kernel-Power → 시스템 비정상 종료, 전원부/과열 의심")과 원문 표시

## 10. 롬라이팅 (ROM Writing)

- **CH341A / CH347 프로그래머 지원** — 드라이버 설치 버튼, 장치 연결상태 표시, 새로고침
- **SPI NOR Flash 읽기 / 쓰기 / 검증** — 열기 · 저장 · 지우기(Chip Erase) · 읽기 · 자동(지우기+쓰기+검증)
- 제조사/파트명 자동감지 + 칩 수동 추가, 칩 정보(용량·페이지·SPI·ID) 표시
- **칩 전압 수동 선택** — 3.3V / 1.8V
- **HEX 데이터 뷰어**(복사 지원) + 장비 연결 가이드(소켓/디버그 1번 Pin 위치 사진)
- 쓰기 후 검증 옵션, 자사 서비스 환경 맞춤 UI, 칩리스트 XML 기반 관리
